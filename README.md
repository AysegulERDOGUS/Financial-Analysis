# Financial-Analysis

In this training, yearly budget and actual data were modeled and organized for analysis purposes. The process focused on understanding the data structure, reviewing the relationships between tables, and analyzing the data together rather than individually. As a result, clear and easy-to-understand results were produced to support finance-focused budget and actual comparisons.
--------------------------------------------------------------------------------------------------------------
Data from different Excel tables was cleaned, organized, and combined to create accurate analysis results. During the data modeling process, the Star Schema approach was used, and tables were separated into fact and dimension tables.

To keep the analysis process organized and protect data, some tables were hidden using authorization settings. The analysis was continued using the prepared datasets while keeping the original raw data unchanged.

In this project, the focus was not only on organizing data but also on understanding the business needs behind the data. Product sales performance, profit margins, and cost structures were analyzed to support budget planning and decision-making processes.

The analysis included:

Calculating product-based profit margins.
Reviewing sales performance and cost changes.
Identifying possible actions to reach fourth-quarter budget targets.
